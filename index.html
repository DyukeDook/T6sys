วิธีเพิ่มหน้า HTML ถัดไป

ส่งไฟล์ HTML ระบบถัดไปมาให้ แล้วเพิ่มเข้าโครงนี้ได้เลย โดยใช้ pattern เดิม

1) วางไฟล์นำไฟล์ไปไว้ใน pages/ชื่อไฟล์.html
2) เพิ่ม metadataแก้ data/pages.json ใส่ title, description, path, category, status, updated
3) เปิดหน้าหลักindex.html จะดึงรายการมาแสดงเป็นการ์ดอัตโนมัติ
4) Deployเปิด GitHub Pages จาก branch main และ folder /root
