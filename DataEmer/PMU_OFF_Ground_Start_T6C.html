ภาพรวม 1 นาที

ใช้เมื่อไหร่ใช้เฉพาะกรณี ferry ไปยังสถานที่ที่ทำ maintenance ได้ เมื่อ autostart ไม่สามารถทำได้
สิ่งที่เปลี่ยนPMU ถูกปิด จึงไม่ควบคุม engine functions และไม่มี auto abort protection
สิ่งที่ pilot ต้องทำควบคุม PCL อย่างระมัดระวัง และ monitor ITT, N1, torque ให้อยู่ใน limit
CAUTION
เมื่อ PMU OFF การ start อยู่ใน manual mode: ต้องถือว่าไม่มีระบบช่วยตัด fuel / igniter / starter เพื่อป้องกัน hot หรือ hung start แบบอัตโนมัติ

Logic ที่ต้องเข้าใจก่อนทำ

PMU switch OFF→ Manual mode→ PCL schedule fuel directly→ Pilot monitors limits→ Abort manually if abnormal
Normal autostartPMU OFF ground start
PMU controls start sequence and can auto abort when it detects no start / hung / hot start trend.PMU is disabled. Pilot must manually manage PCL and abort if indications are unsafe.
Start protection depends on PMU logic.No auto abort protection. The critical monitor is ITT response, N1 acceleration, torque, and engine limits.

Procedure Flow แบบสรุป

  1. Secure aircraft / cockpit
    Canopy closed and locked. Navigation and anti-collision lights as required.
  2. Put engine control into manual mode
    PMU switch OFF. Verify PMU FAIL warning and PMU STATUS caution illuminate.
  3. Configure for manual start
    BLEED AIR INFLOW OFF, BOOST PUMP ON, IGNITION ON, propeller area clear, STARTER switch MANUAL.
  4. Introduce fuel carefully
    At minimum 13% N1, advance PCL only as far as necessary to obtain fuel flow indications. Expect approximately 70-80 pph.
  5. Monitor start
    Monitor ITT and N1 continuously. With PMU disabled, also monitor torque and keep parameters within limits.
  6. Stabilize and reach idle
    Check oil and hydraulic pressure. Advance PCL slowly to reach IDLE at approximately 67% N1.
  7. Return switches to normal post-start configuration
    STARTER NORM, IGNITION NORM, BOOST PUMP ARM, disconnect external power if used and secure door, GEN ON, BLEED AIR INFLOW NORM, then perform Before Taxi Check.
WARNING
Canopy ต้อง closed and locked ก่อน engine start เพราะ exhaust และ propwash อาจทำให้บาดเจ็บหรือเกิดความเสียหายได้

Abort / Stop Start Criteria ที่ต้องจำ

สัญญาณที่เห็นAction / ความหมาย
No rise in ITT ภายใน 10 วินาทีหลังมี fuel flow indicationMove PCL to OFF แล้วทำ Abort Start Procedure
ITT trend มีแนวโน้มเกิน limit / overtemperatureManual abort ทันทีตาม Abort Start Procedure; PMU OFF ไม่มี auto abort
N1 acceleration หยุดหรือผิดปกติสงสัย hung start - abort ตาม procedure
Parameter ใด ๆ เกินหรือกำลังจะเกิน engine limitsลดความเสี่ยง engine damage ด้วยการ abort และทำ motoring run หลัง fuel ถูก introduce แล้ว
CAUTION
หลัง aborted start ที่มีการ introduce fuel แล้ว ให้ทำ Motoring Run Procedure เพื่อ clear residual fuel และ/หรือช่วยลด ITT ก่อนพิจารณา start ครั้งถัดไปตามคู่มือ

Critical Cautions

CAUTION
Starting ใน manual mode ปิดการควบคุม engine functions โดย PMU รวมถึง auto abort
CAUTION
ถ้าไม่มี ITT rise ภายใน 10 วินาทีหลัง fuel flow indication: PCL OFF และ Abort Start Procedure
CAUTION
อย่า push PCL ไป IDLE ก่อน 60% N1 ระหว่าง manual start เพราะอาจทำให้ engine damage จาก overtemperature
Memory Aid: “OFF = Own Fuel Flow”
PMU OFF แล้ว pilot ต้องคุม fuel/monitor limit เอง

Quick Check

1) PMU OFF ground start ใช้เมื่อไหร่?
เมื่อ autostart ไม่สามารถทำได้ และใช้สำหรับ ferry ไปยังสถานที่ที่สามารถทำ maintenance ได้ตามคู่มือ
2) ทำไม procedure นี้เสี่ยงกว่า autostart?
เพราะ PMU ไม่ควบคุม engine functions และไม่มี auto abort protection; pilot ต้อง monitor และ abort เอง
3) ถ้ามี fuel flow แล้ว ITT ไม่ rise ภายใน 10 วินาทีต้องทำอะไร?
PCL OFF แล้วทำ Abort Start Procedure
4) ทำไมต้องระวัง PCL ก่อน 60% N1?
การเอา PCL เข้า IDLE ก่อน 60% N1 ใน manual start อาจทำให้ engine damage เนื่องจาก overtemperature
