21. Abort Start Procedure

ภาพรวมสั้น

Abort start ใช้หยุด engine start เมื่อ PMU/นักบินเห็น hot, hung, no-start, battery bus, หรือ ST READY/PCL logic ผิดปกติ แล้วต้อง motor engine เพื่อ clear fuel/ลด ITT.

Key Facts

KeyAUTO start PMU monitor N1, ITT, fuel flow และสามารถ abort start ขณะ ST READY logic ยัง valid
KeyManual abort จำเป็นเมื่อ hot/hung/no start หรือ ST READY/PCL conditions ผิดปกติ
Keyถ้า fuel introduced แล้วต้อง perform Motoring Run Procedure
KeyRepeated PMU aborts indicate engine malfunction

ระบบที่เกี่ยวข้อง

  • PMU and Auto Start logic
  • STARTER switch AUTO/RESET, NORM, MANUAL
  • PCL start-ready position/ST READY advisory
  • Ignition system/IGN SEL
  • Electric boost pump and starter relay
  • Battery/BATT BUS/MFD ability to monitor start

Mental Model

Abort fuel/ignition now - Motor 20 sec - Cool/report before retry.

Boldface / Critical Action

  1. PCL - OFF; or STARTER switch - AUTO/RESET

Expanded Checklist Flow / ขั้นตอนตาม Manual แบบสรุป

  1. Abort trigger recognized
  2. PCL - OFF; or STARTER switch - AUTO/RESET
  3. Perform Motoring Run Procedure
  4. Motoring Run: PCL OFF; IGNITION NORM; propeller area clear; STARTER MANUAL for 20 sec; STARTER NORM
  5. Observe starter duty cycle/cool-down

Warning / Caution / Note ที่ต้องเน้น

CAUTION

  • If external power start aborted by PMU or manual for hot/hung/no start, do not attempt subsequent starts with same external power cart
  • Repeated PMU aborted starts indicate engine malfunction
  • STARTER switch is not spring-loaded from MANUAL to NORM

NOTE

  • If start initiated with PCL OFF, abort by STARTER AUTO/RESET; if PCL out of OFF but not past IDLE gate, use PCL OFF or AUTO/RESET; if past IDLE gate, PCL OFF
  • Note/report degree and duration of any overtemperature
