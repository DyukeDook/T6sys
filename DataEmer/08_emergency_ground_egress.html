ภาพรวมสั้น

Ground egress ต้องทำให้ ejection seat safe ก่อน ออกทาง canopy ปกติถ้าได้ และใช้ CFS เมื่อ canopy เปิดไม่ได้หรือจำเป็นต้องออกด้านขวา.

Source scope: Section I Canopy Fracturing System and Emergency Ground Egress System; Section III Emergency Ground Egress; Figure 3-13 Critical Action Table.

Key Facts

KeyCFS ใช้ explosive cords เพื่อแตก transparency สำหรับ ejection/egress
KeyISS mode ต้อง SOLO เพื่อป้องกัน inadvertent ejection ของอีก seat
KeySeat safety pins ต้อง install ก่อนออกจาก cockpit
KeyCFS internal handle แต่ละ cockpit แตกเฉพาะ transparency ของตัวเอง; external handle แตกทั้งสอง

ระบบที่เกี่ยวข้อง

  • Interseat Sequencing System (ISS)
  • Ejection seats and seat safety pins
  • Canopy and internal/external opening handles
  • Canopy Fracturing System (CFS) internal/external handles
  • BAT/GEN/AUX BAT electrical switches
  • Harness/lap belt/SSK/leg restraint/oxygen/comm/anti-G connections

Mental Model

SOLO - Pins - Brake - Canopy/CFS - Release - Power off - Out.

ใช้เป็นคำจำเพื่อจัดลำดับความคิด ไม่ใช้แทน checklist จริง

Boldface / Critical Action

รายการนี้อยู่ใน Figure 3-13 Critical Action Table ของ Section III

  1. ISS mode selector - SOLO
  2. Seat safety pin - Install (BOTH)
  3. PARKING BRAKE - As required
  4. Canopy - Open
  5. If canopy cannot be opened or right-side egress required: CFS - Rotate 90° counterclockwise and pull (BOTH)
  6. Shoulder straps, lap belt, SSK fittings, and leg restraint garters - Release (BOTH)
  7. BAT, GEN, and AUX BAT switches - OFF
  8. Evacuate aircraft

Expanded Checklist Flow / ขั้นตอนตาม Manual แบบสรุป

  1. ใช้ canopy normal opening ก่อนถ้าทำได้
  2. ถ้า CFS ใช้ใน cockpit ให้ rotate 90° counterclockwise แล้ว pull
  3. ปลด shoulder straps, lap belt, SSK fittings และ leg restraint garters
  4. สาย oxygen/comm/anti-G ส่วนใหญ่ pull free ขณะออก แต่ต้องไม่ติดขัด
  5. ออกจาก aircraft ไป upwind/ปลอดภัยตามสถานการณ์หน่วย

Warning / Caution / Note ที่ต้องเน้น

WARNING

  • หากไม่ตั้ง ISS SOLO อาจทำให้ one/both seats eject โดยไม่ตั้งใจ
  • หากไม่ใส่ seat safety pins ก่อน egress อาจเกิด inadvertent ejection และ injury/death
  • ก่อน actuate CFS ให้ oxygen mask on และ visor down

CAUTION

  • ถ้า CFS และ canopy latch failure พร้อมกันและ canopy locked อาจเหลือ ejection เป็นทางออก

NOTE

  • Ejection system มี 0/0 capability ในสถานการณ์ ground emergency

Quick Check / Quiz

ทำไม ISS ต้อง SOLO?

เพื่อป้องกันการสั่ง eject อีก seat โดยไม่ตั้งใจ

CFS internal handle แตก canopy ส่วนใด?

เฉพาะ transparency ของ cockpit นั้น; external handle แตกทั้งสอง

อะไรต้องทำก่อน CFS เพื่อป้องกัน injury?

ใส่ oxygen mask และ visor down
