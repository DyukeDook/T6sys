ภาพรวมสั้น

อาการ power/prop ไม่เป็นไปตาม PCL อาจมาจาก PMU scheduling, contamination, prop dump solenoid หรือ oil pressure loss; ขั้นแรกตั้ง PCL mid range แล้ว PMU OFF เพื่อกลับ manual control/prop control logic.

Source scope: Section I PMU/Propeller/PIU; Section III Uncommanded Power Changes/Loss of Power/Uncommanded Propeller Feather; Figure 3-13.

Key Facts

KeySymptoms: uncommanded thrust reduction, lower fuel flow/N1, step change when PMU drops offline
KeyUncommanded feather: rapid NP decrease + high torque + noise/vibration/loss of thrust
KeyPROP SYS CB pull removes power from feather dump solenoid if NP stable below 40%
KeyManual mode increases oil pressure/volume to prop pitch control and may restore prop control

ระบบที่เกี่ยวข้อง

  • PMU and manual mode
  • Propeller Interface Unit (PIU) and oil pressure to prop pitch mechanism
  • Feather dump solenoid and PROP SYS circuit breaker
  • PCL/FMU fuel scheduling
  • Engine indications: N1, torque, ITT, fuel flow, NP
  • PEL/forced landing/ejection decision

Mental Model

Mid-range - PMU off - Prop CB if low stable NP - assess power - PEL or shutdown/eject.

วิธีจัดการ / Management Logic

PCL mid range gives recovery chance while limiting overtorque/overtemp when PMU OFF
PMU OFF removes faulty scheduling and shifts to manual mode
If NP stable below 40%, pull PROP SYS CB to remove feather dump solenoid power
Set PCL as required and assess power sufficiency
If power sufficient execute PEL; if insufficient, reset PROP SYS as required, shutdown, forced landing or eject

Boldface / Critical Action

รายการนี้อยู่ใน Figure 3-13 Critical Action Table ของ Section III

  1. PCL - MID RANGE
  2. PMU SWITCH - OFF
  3. PROP SYS CIRCUIT BREAKER (left front console) - PULL, IF NP STABLE BELOW 40%
  4. PCL - As required
  5. If power sufficient: PEL - Execute
  6. If power insufficient: PROP SYS CB - Reset as required
  7. PCL - OFF
  8. FIREWALL SHUTOFF handle - Pull
  9. Execute Forced Landing or Eject

Expanded Checklist Flow / ขั้นตอนตาม Manual แบบสรุป

  1. PCL - MID RANGE
  2. PMU SWITCH - OFF
  3. If NP stable below 40%: PROP SYS CB - PULL
  4. PCL - As required
  5. Assess descent rate/power available
  6. If power sufficient: PEL
  7. If power insufficient: reset PROP SYS CB as required; PCL OFF; FIREWALL SHUTOFF pull; forced landing/eject

Warning / Caution / Note ที่ต้องเน้น

WARNING

  • Potential ITT limit exceedance if PMU OFF with ITT ≥820°C
  • With PROP SYS CB pulled and PMU OFF, propeller may feather slower; glide performance may be considerably reduced

CAUTION

  • Ground idle will not be available for landing rollout/taxi with PMU OFF; plan increased landing distance due to higher idle N1
  • If engine is underpowered and descent rates are high, delaying shutdown to feather may leave insufficient altitude

NOTE

  • Mid-range PCL is physical midpoint between IDLE and MAX
  • Propeller should come out of feather within 15-20 sec after PROP SYS CB action if electrical signal was cause
  • PMU cycle NORM then OFF may restore NP/torque display if red X caused by RPM rate limit exceedance
