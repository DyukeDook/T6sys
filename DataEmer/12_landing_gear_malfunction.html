ภาพรวมสั้น

ใช้ checklist นี้เมื่อ gear ไม่แสดง fully up/down ตาม handle; ต้องแยก unsafe indication, hydraulic problem, light/CB problem และ gear/door position จริง.

Source scope: Section I Landing Gear System/Controls and Indicators; Section III Landing Gear Malfunction.

Key Facts

Keyใช้เมื่อ gear handle up แล้ว gear ไม่ fully up หรือ handle down แล้ว gear ไม่ fully down
KeySafe gear-down indications อาจยืนยันจาก combined cockpit lights, HUD cues, landing/taxi lights
KeyVisual inspection by chase/RDO/tower preferred ถ้ามี
Keyถ้า hydraulic pressure <1800 psi ให้ใช้ emergency extension

ระบบที่เกี่ยวข้อง

  • Hydraulic gear actuators and main gear door actuator
  • Landing gear handle and red/green position lights
  • LDGGR CONT, INST, INST LT circuit breakers
  • LAMP TEST
  • Gear warning tone/HUD long horizon/E-bracket
  • Emergency LDG GR handle

Mental Model

Slow - Down - Lamp - Pressure - CB - Cycle - Confirm - Choose unsafe/emergency.

ใช้เป็นคำจำเพื่อจัดลำดับความคิด ไม่ใช้แทน checklist จริง

Boldface / Critical Action

ไม่พบเป็นรายการ Critical Action/Boldface ใน Figure 3-13 สำหรับหัวข้อนี้ ให้ปฏิบัติตาม checklist Section III และคำสั่ง instructor/หน่วยบินตามลำดับ

Expanded Checklist Flow / ขั้นตอนตาม Manual แบบสรุป

  1. Airspeed - Remain below 150 KIAS
  2. Gear handle - DOWN, press firmly; BOTH
  3. LAMP test switch - Check
  4. Hydraulic pressure - Check; if below 1800 psi execute emergency extension
  5. LDGGR CONT, INST, INST LT CBs - Check in/reset
  6. Gear handle - Cycle as appropriate
  7. Gear and gear door positions - Check visually if able
  8. If specified unsafe conditions remain: Landing with Unsafe Gear Indications
  9. If none of those conditions remain: Landing Gear Emergency Extension

Warning / Caution / Note ที่ต้องเน้น

CAUTION

  • อย่า delay visual inspection coordination ถ้ามี aircraft/RDO/tower available
  • หลาย gear cycles ทำได้จนถึง critical fuel หรือ PIC เห็นว่าเสี่ยง damage/hydraulic pressure loss

NOTE

  • ถ้า main gear down/locked และ inboard doors closed, nose gear can be assumed down/locked
  • Nose-only unsafe อาจเป็น downlock microswitch fault; LDGGR CONT reset อาจช่วยได้
