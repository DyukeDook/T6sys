07. Trim Aid Device Failure (TAD)

TAD FAIL หมายถึง rudder trim aid ไม่คำนวณ/ป้อน correction ให้ yaw trim; workload เพิ่มแต่ mission capability ไม่เสียโดยตรง.

ภาพรวมสั้น

TAD FAIL หมายถึง rudder trim aid ไม่คำนวณ/ป้อน correction ให้ yaw trim; workload เพิ่มแต่ mission capability ไม่เสียโดยตรง.

Source scope: Section I Trim Aid Device; Section III Trim Aid Device Failure; Increased Rudder Pedal Forces.

Key Facts

KeyTAD ใช้ torque, altitude, airspeed, pitch rate เพื่อคำนวณ rudder trim tab position
KeyTAD OFF เป็น advisory; TAD FAIL เป็น caution
KeyManual yaw trim input additive กับ TAD command
KeyTAD failure เพิ่ม workload แต่ไม่มีผลอื่นต่อ mission capability ตาม manual

ระบบที่เกี่ยวข้อง

  • TAD computer/switch บน left console front cockpit
  • Rudder trim tab actuator
  • RUD TRIM power and TAD circuit breaker
  • EICAS TAD OFF/TAD FAIL
  • Turn-and-slip indicator/rudder pedal force

Mental Model

TAD FAIL = yaw helper lost: reset once, manual trim, tell maintenance.

ใช้เป็นคำจำเพื่อจัดลำดับความคิด ไม่ใช้แทน checklist จริง

Boldface / Critical Action

ไม่พบเป็นรายการ Critical Action/Boldface ใน Figure 3-13 สำหรับหัวข้อนี้ ให้ปฏิบัติตาม checklist Section III และคำสั่ง instructor/หน่วยบินตามลำดับ

Expanded Checklist Flow / ขั้นตอนตาม Manual แบบสรุป

  1. TAD circuit breaker - Check in or pull and reset
  2. Wait minimum 15 seconds before reset
  3. TRIM AID switch - Engage after TAD FAIL extinguishes
  4. Notify maintenance after landing even if reset successful

Warning / Caution / Note ที่ต้องเน้น

NOTE

  • TAD will not completely trim yaw; pilot ยังต้อง trim/manual rudder
  • ระหว่าง torque/airspeed/pitch rate changes อาจรู้สึก pedal movement จาก TAD ได้ในสภาพปกติ
