  • Martin-Baker ejection seat/catapults/rocket motor
  • CFS automatic initiation during ejection
  • ISS selector for solo/dual sequencing
  • Emergency oxygen, MOR handle, parachute and seat separation

Mental Model

Aviate to survive: decide early, position if able, pull handle.

Boldface / Critical Action

รายการนี้อยู่ใน Figure 3-13 Critical Action Table ของ Section III

  1. EJECTION HANDLE - PULL (BOTH)

Expanded Checklist Flow / ขั้นตอนตาม Manual แบบสรุป

  1. Controlled ejection prep as time permits: notify crew, 2000 ft AGL minimum recommended, 125-180 KIAS recommended
  2. Transmit distress call and set transponder 7700 if time permits
  3. Stow loose equipment; visor down; oxygen mask/helmet fastened; leg restraints checked; harness locked
  4. PCL - OFF if time and conditions permit
  5. Assume proper position: head back, shoulders/back against seat, elbows in, legs flat/extended
  6. Execute EJECT

Warning / Caution / Note ที่ต้องเน้น

WARNING

  • ถ้า aircraft uncontrollable ให้ eject regardless of speed, altitude, or attitude เพราะ immediate ejection ให้โอกาสรอดสูงสุด
  • อย่า delay decision to eject; pilot reaction time, attitude, airspeed, sink rate มีผลต่อ safe altitude
  • CFS fragments/hot shards: exposed skin covered, mask on, visor down

CAUTION

  • Over mountainous terrain >8000 ft MSL ควรใช้ MOR เพื่อ manually separate/deploy parachute

NOTE

  • Low-speed ejection อาจทำให้ risers ยัง velcroed; manually separate if time permits
