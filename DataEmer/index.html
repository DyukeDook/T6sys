Index: 23 Emergency HTML Files

สรุปแยกหัวข้อละ 1 HTML ตามคำขอ: ระบบที่เกี่ยวข้อง, analyze indicators, management และ boldface/critical action

Training aid สำหรับทบทวน Emergency/Abnormal Procedure ของ T-6C เท่านั้น ไม่ใช้แทน Flight Manual, checklist, instructor guidance หรือคำสั่งหน่วยบิน

รายการไฟล์

01. LOW FUEL PRESSURE

Fuel feed pressure ต่ำกว่าเกณฑ์จนระบบแจ้ง FUEL PX; ต้องแยกให้ออกว่า boost pump ช่วยได้หรือเป็นปัญหา fuel feed/engine-driven pump/fuel leak.

เปิด HTML
02. FUEL IMBALANCE

FUEL BAL caution คือ fuel ซ้าย-ขวาต่างเกิน logic ของระบบ หรือมี probe/EDM fault; ต้องยืนยัน imbalance จริงก่อน manual balance.

เปิด HTML
03. Inadvertent Departure from Controlled Flight

OCF/departure ต้องหยุดพลังงานที่ทำให้อาการรุนแรง Neutralize controls เช็ค altitude แล้วค่อย recover จาก unusual attitude.

เปิด HTML
04. Canopy Unlocked

CANOPY warning/unsafe canopy in flight คือปัญหา latch/lock/seal ที่ต้องลด airspeed ห้ามพยายาม lock in flight ลด altitude/pressure แล้ว land as soon as possible.

เปิด HTML
05. Runaway Trim

Runaway trim คือ trim input เคลื่อนไปเองหรือค้างจน control force เพิ่ม; ต้องหยุด trim power ด้วย interrupt/disconnect แล้ว isolate axis ที่ผิด.

เปิด HTML
06. Stuck/Failed Trim

Stuck/failed trim คือ trim ไม่ตอบสนองหรือค้างตำแหน่ง; ต้องยืนยัน disconnect อยู่ NORM ตรวจ CB แล้วบินด้วย control force ที่เพิ่มขึ้นตาม axis.

เปิด HTML
07. Trim Aid Device Failure (TAD)

TAD FAIL หมายถึง rudder trim aid ไม่คำนวณ/ป้อน correction ให้ yaw trim; workload เพิ่มแต่ mission capability ไม่เสียโดยตรง.

เปิด HTML
08. Emergency Ground Egress

Ground egress ต้องทำให้ ejection seat safe ก่อน ออกทาง canopy ปกติถ้าได้ และใช้ CFS เมื่อ canopy เปิดไม่ได้หรือจำเป็นต้องออกด้านขวา.

เปิด HTML
09. Eject

Eject เป็น action สุดท้ายเมื่อ aircraft/landing option ไม่ปลอดภัย; ถ้าควบคุมได้ให้ทำ controlled ejection prep แต่เมื่อไม่มีเวลาให้ pull handle ทันที.

เปิด HTML
10. Post Ejection Procedures

หลัง seat separation ต้องตรวจ canopy/lines ควบคุม parachute เตรียม PLF หรือ water entry และใช้ survival equipment ให้เหมาะกับ terrain.

เปิด HTML
11. Hydraulics System Malfunction

Hydraulic malfunction กระทบ gear/flaps/speed brake/NWS; ต้องอ่าน HYD FL LO/EHYD PX LO/pressure ให้ถูกเพื่อเลือก normal vs emergency extension.

เปิด HTML
12. Landing Gear Malfunction

ใช้ checklist นี้เมื่อ gear ไม่แสดง fully up/down ตาม handle; ต้องแยก unsafe indication, hydraulic problem, light/CB problem และ gear/door position จริง.

เปิด HTML
13. Landing with Unsafe Gear Indications

ถ้า gear ใดไม่ confirmed fully extended ให้ปฏิบัติเหมือน gear นั้น retracted และวางแผน flat, power-on, straight-in landing.

เปิด HTML
14. Landing Gear Emergency Extension

Emergency extension ใช้ emergency accumulator/lines ขยาย gear/flaps เมื่อ normal system unreliable; หลัง pull handle gear/flaps retract ไม่ได้.

เปิด HTML
15. Leaking Fuel from Wing

Fuel leak จาก wing ใช้ manual balance logic เพื่อใช้ fuel จาก leaking tank ให้หมดก่อน ลดการสูญเสีย total fuel แล้ว land as soon as possible.

เปิด HTML
16. Fuel Probe Malfunction

FP FAIL ทำให้ fuel quantity/imbalance indication อาจผิดและ auto balance unavailable; ต้อง verify fuel state ด้วย gages/fuel flow และห้าม manual balance โดยอาศัย indication ที่ผิด.

เปิด HTML
17. High Fuel Flow

Fuel flow 800 PPH or greater เป็นสัญญาณ fuel system malfunction และอาจนำไป engine failure; checklist ให้ execute PEL ทันที.

เปิด HTML
18. OBOGS Fail Message

OBOGS FAIL อาจมาจาก low bleed pressure, concentration/pressure failure, leak หรือ electrical/BIT; ถ้ามี symptoms ให้ไป physiological checklist ทันที.

เปิด HTML
19. OBOGS Failure/Physiological Symptoms

เมื่อสงสัย hypoxia/toxin/OBOGS failure ให้ใช้ emergency oxygen ทันที ลด altitude ต่ำกว่า 10,000 ft MSL ก่อน oxygen หมด และ land as soon as possible.

เปิด HTML
20. Environmental Systems Duct Overtemp

DUCT TEMP คือ bleed air temperature ใน ECS/defog duct เกิน 300°F; ต้องสั่งระบบไป cold ลด defog/bleed และลด altitude ตาม pressurization need.

เปิด HTML
21. Abort Start Procedure

Abort start ใช้หยุด engine start เมื่อ PMU/นักบินเห็น hot, hung, no-start, battery bus, หรือ ST READY/PCL logic ผิดปกติ แล้วต้อง motor engine เพื่อ clear fuel/ลด ITT.

เปิด HTML
22. Emergency Engine Shutdown on the Ground

Ground serious emergency ต้องตัด power/fuel/hydraulic/bleed at firewall แล้ว egress as required เพื่อป้องกัน fire, prop/aircraft movement, หรือ damage ต่อ crew/aircraft.

เปิด HTML
23. Uncommanded Power Changes/Loss of Power/Uncommanded Propeller Feather

อาการ power/prop ไม่เป็นไปตาม PCL อาจมาจาก PMU scheduling, contamination, prop dump solenoid หรือ oil pressure loss; ขั้นแรกตั้ง PCL mid range แล้ว PMU OFF เพื่อกลับ manual control/prop control logic.

เปิด HTML
