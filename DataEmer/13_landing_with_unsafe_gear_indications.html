ภาพรวมสั้น

ถ้า gear ใดไม่ confirmed fully extended ให้ปฏิบัติเหมือน gear นั้น retracted และวางแผน flat, power-on, straight-in landing.

Source scope: Section III Landing with Unsafe Gear Indications; Landing Gear Malfunction; Figure 3-12.

Key Facts

KeyGear-up landing to suitable landing area preferred if any gear confirmed unsafe
KeyTreat any landing gear not fully extended as retracted
KeyFuel reduce recommended no less than 100 lb
Keyหลังหยุด ถ้า light ยัง unsafe ห้าม taxi/tow จนใส่ gear safety pins

ระบบที่เกี่ยวข้อง

  • Landing gear configuration and door positions
  • Hydraulic gear/flap system
  • PCL/engine shutdown and firewall shutoff
  • OBOGS/anti-suffocation valve after engine shutdown
  • CFS/canopy considerations after abnormal landing

Mental Model

Unsafe gear = assume retracted; flat power-on straight-in; shutdown when assured.

ใช้เป็นคำจำเพื่อจัดลำดับความคิด ไม่ใช้แทน checklist จริง

Boldface / Critical Action

ไม่พบเป็นรายการ Critical Action/Boldface ใน Figure 3-13 สำหรับหัวข้อนี้ ให้ปฏิบัติตาม checklist Section III และคำสั่ง instructor/หน่วยบินตามลำดับ

Expanded Checklist Flow / ขั้นตอนตาม Manual แบบสรุป

  1. Gear handle - Raise if able
  2. Fuel - Reduce; recommend no less than 100 lb
  3. Flaps - As required; recommend LDG
  4. Harness - Locked (BOTH)
  5. Execute landing technique for configuration
  6. When landing assured: PCL - OFF
  7. FIREWALL SHUTOFF handle - Pull
  8. After stop: Emergency Ground Egress as required

Warning / Caution / Note ที่ต้องเน้น

WARNING

  • A gear-up landing to suitable landing area is preferred if any gear is confirmed unsafe
  • Treat any landing gear not fully extended as retracted

CAUTION

  • Main gear only: do not let nose forcefully strike ground; could damage structure/CFS/canopy opening
  • Engine shutdown reduces drag because prop feathers; be ready for balloon in flare

NOTE

  • Be prepared to use anti-suffocation valve when OBOGS shuts down with engine
