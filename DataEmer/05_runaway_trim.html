05. Runaway Trim

ภาพรวมสั้น

Runaway trim คือ trim input เคลื่อนไปเองหรือค้างจน control force เพิ่ม; ต้องหยุด trim power ด้วย interrupt/disconnect แล้ว isolate axis ที่ผิด.

Source scope: Section I Flight Control System/Trim Aid Device; Section III Trim System Malfunctions - Runaway Trim.

Key Facts

KeyTrim interrupt button ตัด power ทุก trim actuators และ disengage TAD
KeyTRIM DISCONNECT ตัด trim system และทำให้ TAD disengage
KeyAirspeed มีผลต่อ control force
Keyใช้ CB เพื่อ isolate AIL/EL หรือ RUD trim ตาม axis ที่ผิด

ระบบที่เกี่ยวข้อง

  • Aileron/elevator trim switch บน control stick
  • Rudder trim switch บน PCL
  • Trim interrupt button บน stick
  • TRIM DISCONNECT switch บน left console
  • Triple trim indicator และ AIL/EL TRIM/RUD TRIM circuit breakers
  • TAD จะหลุดเมื่อ trim interrupt หรือ trim disconnect

Mental Model

Interrupt - Slow/adjust - Disconnect - Isolate CB - Restore good axes.

ใช้เป็นคำจำเพื่อจัดลำดับความคิด ไม่ใช้แทน checklist จริง

Boldface / Critical Action

ไม่พบเป็นรายการ Critical Action/Boldface ใน Figure 3-13 สำหรับหัวข้อนี้ ให้ปฏิบัติตาม checklist Section III และคำสั่ง instructor/หน่วยบินตามลำดับ

Expanded Checklist Flow / ขั้นตอนตาม Manual แบบสรุป

  1. Trim interrupt button - Depress and hold
  2. Airspeed - As required to reduce control forces
  3. TRIM DISCONNECT switch - TRIM DISCONNECT
  4. Trim interrupt button - Release
  5. AIL/EL TRIM or RUD TRIM CB(s) - Pull as required
  6. TRIM DISCONNECT switch - NORM
  7. Controllability check - As required

Warning / Caution / Note ที่ต้องเน้น

NOTE

  • โดยทั่วไป 110-150 KIAS ลด control forces ยกเว้น full nose-down trim
  • Power change ส่งผล pitch/yaw: add power = pitch up/left yaw; reduce power = pitch down/right yaw
  • ใน aileron trim malfunction final approach force มักยังเบา จึงอาจทิ้ง AIL/EL CB ไว้เพื่อใช้ pitch trim ถ้าจำเป็น

Quick Check / Quiz

ปุ่ม trim interrupt ทำอะไร?

ตัด power ให้ trim actuators ทุกแกนและ disengage TAD ขณะกดค้าง

หลัง pull CB แล้วทำไมอาจ return TRIM DISCONNECT to NORM?

เพื่อคืน trim ที่ยังใช้งานได้ในแกนอื่น

ทำไม airspeed จึงสำคัญ?

เพราะ aerodynamic load บน control surface/trim มีผลต่อ control force
