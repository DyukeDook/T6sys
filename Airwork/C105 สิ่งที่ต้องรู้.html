T-6C Airstart & Ground Handling Study Notes

สรุปเพื่อเข้าใจ logic ของ 4 หัวข้อ: PMU Normal Air Start, PMU OFF Air Start, Immediate Air Start และ Blown Tire

๑.๑ PMU Normal Air Start

Primary airstartPMU normalLeast complicatedStarter assisted

PMU NORM airstart คือวิธี หลัก และเป็นวิธีที่ง่ายที่สุดเมื่อ PMU ยังปกติอยู่ ใช้สำหรับกรณี engine failure ที่ ไม่ได้เกิดจาก fire หรือ mechanical failure โดยระบบ start ยังอาศัย starter assist

Mental model: ถ้า PMU ยังใช้ได้ ให้ใช้ PMU NORM airstart เพราะระบบจะช่วยจัดการ start ให้มากที่สุด เหลือนักบินคุมลำดับและ monitor engine indications
Airstart envelope: manual ระบุว่า airstart ควรพยายามทำภายใน envelope ที่แนะนำ — โดยคร่าว ๆ 125-200 KIAS ตั้งแต่ sea level ถึง 15,000 ft และ 135-200 KIAS ที่ 15,001-20,000 ft และ airstart not recommended below 2000 ft AGL

Core checklist logic

  1. PCL - OFF ต้อง OFF เพื่อ feather prop และให้ starter/ignition/boost pump/PMU ทำงานถูกต้อง
  2. Confirm circuit breakers and firewall shutoff handle START, IGN, BOOST PUMP, PMU CBs in; firewall handle down
  3. BLEED AIR INFLOW - OFF
  4. BOOST PUMP - ON
  5. IGNITION - ON
  6. STARTER - AUTO/RESET
  7. PCL - IDLE above 13% N1
  8. Monitor ITT, N1, oil pressure
  9. If successful: PCL as required after N1 reaches idle (~67%), then restore systems and execute PEL
  10. If unsuccessful: PCL OFF, firewall shutoff handle pull, then forced landing or eject
จุดสำคัญเหตุผลถ้าพลาดจะเกิดอะไร
PCL ต้อง OFF ตอนเริ่มป้องกัน fuel เข้าเร็วเกินก่อนระบบพร้อมfuel introduced prematurely, start พังหรือร้อนเกิน
เอา PCL ไป IDLE เฉพาะเมื่อ N1 > 13%ให้ core speed พอสำหรับ light-off ที่เหมาะสมstart อาจไม่ติดหรือไม่เสถียร
ห้ามขยับ PCL เลย IDLE ก่อน N1 stabilize ~67%ป้องกัน fuel flow พุ่งและ ITT overtempcatastrophic engine damage จาก severe ITT overtemperature
ถ้าไม่มี ITT rise ภายใน 10 วินาทีหลังมี fuel flowบ่งชี้ unsuccessful light-offต้อง abort start ทันที
ข้อห้ามที่ต้องจำ: ถ้า engine failure เกิดจาก fire หรือ mechanical failure ไม่ควรฝืน airstart แบบ routine และถ้าต่ำกว่า 2000 ft AGL อย่าหน่วงเวลาการ eject เพราะมัวแต่พยายาม restart
ภาพรวมหลัง start สำเร็จ: reset/verify GEN, คืน BLEED AIR INFLOW, จัดการ OBOGS ตามต้องการ แล้วเข้าสู่ PEL / recovery plan ทันที
จำง่าย: “PCL OFF — system set — starter AUTO — PCL IDLE above 13% — monitor — ถ้าได้ก็ PEL, ถ้าไม่ได้ก็ land/eject”

๑.๒ PMU OFF Air Start

PMU FAILManual engine start logicMost critical PCL movement

ถ้า PMU FAIL warning ติด แปลว่า PMU malfunction และ ต้องใช้ PMU OFF airstart เพราะ PMU ไม่ช่วยจัดการ start ให้ตามปกติแล้ว ในโหมดนี้ igniters จะไม่ทำงานอัตโนมัติ นักบินต้องสั่งเอง และการขยับ PCL จะเป็นจุดสำคัญที่สุด

Mental model: PMU OFF airstart = start แบบ manual มากขึ้น ต้องคุมทั้ง ignition, starter และ fuel introduction ด้วยมือ
คู่มือระบุชัด: “The most critical pilot action during the PMU OFF start is PCL movement while monitoring fuel flow, ITT, and N1 acceleration.”

Core checklist logic

  1. PCL - OFF
  2. PMU switch - OFF
  3. Confirm START/IGN/BOOST PUMP CBs and firewall handle
  4. BLEED AIR INFLOW - OFF
  5. BOOST PUMP - ON
  6. IGNITION - ON
  7. STARTER - MANUAL
  8. At 13% N1 minimum, advance PCL to obtain initial fuel flow, then slowly to IDLE
  9. Monitor ITT, N1, oil pressure
  10. If successful: PCL as required after N1 idle (~67%), starter NORM, verify GEN ON/reset, bleed air NORM, OBOGS as required, execute PEL
  11. If unsuccessful: PCL OFF, firewall shutoff handle pull, forced landing or eject
เรื่องที่ต้องเข้าใจความหมาย
ทำไมใช้ STARTER MANUALเพราะ PMU ไม่ช่วย sequencing ตามปกติ จึงต้องพึ่ง manual control มากขึ้น
ทำไม PCL ต้องค่อย ๆ ไป IDLEถ้าเร็วไป ITT พุ่ง; ถ้าช้าไป N1 อาจ roll back พร้อม ITT ลดลง
ทำไม boost pump ONเพื่อ positive fuel pressure ระหว่าง start
ทำไมยังต้องคิดเรื่อง altitude lossแต่ละ attempt โดยทั่วไปเสีย altitude มากกว่า 1200 ft ได้ จึงต้องประเมิน margin ก่อนเริ่ม
หลุมพราง: ถ้าสตาร์ตโดย PCL ไม่อยู่ OFF จะมี fuel เข้าก่อน ignition พร้อม ทำให้มีโอกาสเกิด hot start สูงมาก
จำง่าย: “PMU FAIL = PMU OFF — starter MANUAL — PCL ค่อย ๆ เข้า — monitor ใกล้ชิดสุด”

๑.๓ Immediate Air Start (PMU NORM)

Low altitudeTime criticalCondensed procedure

Immediate Airstart (PMU NORM) ใช้หลัง engine failure เมื่ออยู่ต่ำและต้องการ thrust เร็ว หรือเมื่อ ไม่มีเวลา/เงื่อนไขไม่เอื้อ ให้ทำ full airstart procedure ครบทั้งหมด เป็นการย่อขั้นตอนเพื่อรีบให้เครื่องกลับมามีกำลัง

Mental model: Immediate airstart = เวอร์ชันย่อเพื่อซื้อเวลาและ thrust กลับมาเร็วที่สุด แต่ยังต้องรักษา PCL discipline เหมือนเดิม

Core checklist logic

  1. PCL - OFF
  2. STARTER - AUTO/RESET
  3. PCL - IDLE above 13% N1
  4. Monitor ITT, N1, oil pressure
  5. If unsuccessful: PCL OFF, firewall shutoff handle pull, forced landing or eject
  6. If successful: PCL as required after N1 idle (~67%), execute PEL, then confirm boost pump ON, ignition ON, starter NORM, bleed air NORM, GEN verify ON/reset

Immediate airstart ใช้เมื่อไร

  • engine failure หลัง takeoff / low altitude
  • thrust requirements critical
  • เวลาไม่พอทำ full checklist ทั้งหมดก่อน

สิ่งที่ยังเหมือน PMU NORM

  • PCL ต้อง OFF ตอนเริ่ม
  • above 13% N1 ค่อยไป IDLE
  • อย่า overtravel PCL ก่อน N1 stabilize
  • ต่ำกว่า 2000 ft AGL อย่ามัวแต่พยายาม start ถ้าไม่มี margin
จุดอันตราย: ความเร่งรีบทำให้ pilot เผลอลืมว่า restart attempt ก็กินเวลาและ altitude ถ้าสถานการณ์ไม่ได้ให้โอกาสจริง ๆ ต้องหันไป forced landing หรือ eject ให้เร็ว
หลัง start ติด: อย่าคิดว่าจบงาน — ต้อง restore supporting switches ให้ครบ, check generator, then execute PEL / recovery เพราะอาจยังเป็น partial power / unstable engine event ได้
จำง่าย: “Immediate = start ให้เร็วที่สุด แต่ถ้าไม่ได้ต้องตัดใจเร็วที่สุดเหมือนกัน”

๑.๔ Blown Tire

Landing rolloutDirectional controlMain tireDo not taxi with flat tire

Blown tire ใน T-6C มักเกี่ยวข้องกับช่วง landing rollout และ braking technique คู่มือชี้ว่า main tire ที่แตกอาจไม่ชัดเจนทันที สัญญาณแรกอาจเป็นแรงดึงไปด้านยางที่แตก หรือ rumble / shudder ที่อาจถูกเข้าใจผิดว่าเป็น nose wheel shimmy

Mental model: Blown tire = เป้าหมายแรกคือรักษา directional control; เป้าหมายถัดมาคือหยุดอย่างปลอดภัยโดยไม่ทำให้สถานการณ์ลุกลาม
สถานการณ์สิ่งที่คู่มือเน้นเหตุผล
รู้ก่อนลงว่ามี main tire แตกfly straight-in approach และพยายามลงด้าน runway ที่ตรงกับ good tireช่วยลด tendency ที่ aircraft จะ drift ไปทางล้อเสีย
ยางแตกขณะเบรกเครื่องอาจ drift away from the blown tire initiallypilot ต้องพร้อมแก้ทิศทางด้วย rudder, brakes และ nose wheel steering
หลังหยุดDo not taxi with a flat tireลดความเสียหายต่อ gear/wheel และลดความเสี่ยงเสียการควบคุมเพิ่มเติม
1. Recognizeสังเกต pull, rumble, shudder และ directional control difficulty
2. Keep controlใช้ rudder เป็นหลัก เสริมด้วย braking / nosewheel steering ตามความเหมาะสม
3. Leave flaps as isถ้าสงสัย blown main tire ระหว่าง rollout ให้คง flap setting เดิม
4. Be ready for follow-on emergencyถ้าคุมไม่ได้ให้เตรียม Aircraft Departs Prepared Surface / Emergency Ground Egress
5. Stop and assessหยุดให้ปลอดภัยก่อน อย่าพยายาม taxi ต่อด้วย flat tire
6. Think prevention tooยางแตกจำนวนมากเกิดก่อน tire spin-up หรือความเร็วต่ำจากการ hold brake มากเกิน
Prevention note จากคู่มือ: ยางแตกจำนวนมากเกิดก่อน tire spin-up หรือที่ความเร็วต่ำกว่า 40 knots และการ hold brake มากเกินช่วงต่ำกว่า 20 knots อาจทำให้ล้อล็อกจนเกิด blown tire ได้
อีกเรื่องที่เกี่ยวข้อง: T-6C ไม่มี anti-skid/anti-lock protection การเบรกแรงเกิน โดยเฉพาะก่อน weight ลงเต็มล้อหรือที่ความเร็วสูง อาจนำไปสู่ locked brake, blown tire, brake fade และ directional control problem
จำง่าย: “ยางแตก = คุมทิศทางก่อน หยุดให้ปลอดภัย อย่าฝืน taxi”
