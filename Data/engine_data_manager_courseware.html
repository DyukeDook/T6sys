Engine Data Manager (EDM)

ระบบรวบรวมและประมวลผลข้อมูลเครื่องยนต์และระบบประกอบ เพื่อส่งไปแสดงบน EICAS พร้อม crew alerting ที่เกี่ยวข้อง

Memory Aid

EDM = Sensor Hub + Data Processor + EICAS Source

Objectives Overview Data Flow EICAS Failures Notes / Warnings Quiz

Learning Objectives

เมื่อเรียนจบหน้านี้ ผู้เรียนควรอธิบายระบบ EDM ได้ในระดับใช้งาน cockpit และเข้าใจ indication สำคัญเมื่อระบบผิดปกติ

1

บอกหน้าที่ EDM

แยกบทบาทของ EDM ออกจาก PMU และ EICAS ได้

2

อธิบาย Data Flow

Sensor / transducer → EDM → IAC → MFD/EICAS

3

อ่าน Failure Indication

แยก EDM FAIL, EDM A INOP, EDM B INOP และ red X/missing pointer ได้

1. System Overview

EDM ไม่ใช่ตัวควบคุมกำลังเครื่องยนต์หลัก แต่เป็นระบบ monitor / data processing สำหรับ engine และ systems หลายรายการ

“EDM ไม่ได้ขับเครื่อง — EDM บอกเราว่าเครื่องและระบบสำคัญกำลังเป็นอย่างไร”

Engine Functions

EDM monitor engine operating parameters และช่วยให้ EICAS แสดงค่าที่เกี่ยวข้อง เช่น torque, oil pressure, oil temperature, fuel flow, N1 และ ITT

Non-engine Functions

EDM ยังเกี่ยวข้องกับ fuel balancing, fuel quantity indication, DC volts/amps, hydraulic pressure, cockpit pressure altitude และ cockpit differential pressure

2. Normal Data Flow

ภาพรวมการไหลของข้อมูลจาก sensor จนถึง cockpit display

Input

Sensors / Transducers

รับข้อมูลจริงจาก engine และระบบประกอบ เช่น oil, fuel, hydraulic, electrical, pressure

Processor

Engine Data Manager

ประมวลผลข้อมูลและสร้าง output ผ่าน EDM A / EDM B channels

Avionics

IAC

Integrated Avionics Computer รับข้อมูลจาก EDM และส่งต่อให้ระบบ display

Cockpit

MFD / EICAS

แสดง engine, auxiliary instruments และ crew alerting ในแต่ละ cockpit

3. EICAS Indications

ข้อมูล EDM ถูกนำไปแสดงบน EICAS รวมทั้ง warning, caution, status และ advisory messages

หมวดตัวอย่างข้อมูลที่แสดงความหมายสำหรับ Pilot
EngineTorque, fuel flow, N1, ITT, oil pressure, oil temperatureใช้ monitor health และ operating limits ของเครื่องยนต์
FuelFuel quantity, fuel balance, bingo fuelใช้ติดตามปริมาณเชื้อเพลิงและ imbalance
ElectricalDC volts, DC ampsใช้ติดตามสภาพระบบไฟฟ้า
HydraulicHydraulic pressureใช้ติดตามความพร้อมของ hydraulic system
PressurizationCockpit pressure altitude, cockpit differential pressureใช้ติดตาม cockpit pressurization

Red Warning

ต้องดึงความสนใจทันที เช่น EDM FAIL

Amber Caution

ใช้กับ condition ที่ต้องระวัง/จัดการตาม checklist

White Status / Advisory

ใช้กับสถานะ เช่น EDM A INOP หรือ EDM B INOP เมื่ออยู่บนพื้น

4. Failure Modes & Crew Indications

แยกความหมายของ failure message และผลกระทบต่อข้อมูลที่แสดง

Message / Indicationชนิดความหมายการตีความข้อมูล
EDM FAILWarningEngine Data Manager failedให้ถือว่า EDM data unavailable ทั้งหมด
EDM A INOPAdvisory / StatusInternal failure ของ channel A หรือ data bus/channel ที่เกี่ยวข้องinvalid เฉพาะค่าที่มี red X หรือ pointer หาย
EDM B INOPAdvisory / StatusInternal failure ของ channel B หรือ data bus/channel ที่เกี่ยวข้องinvalid เฉพาะค่าที่มี red X หรือ pointer หาย
Red X / missing pointerInvalid data cueค่าที่แสดงไม่ validอย่าใช้ parameter นั้นเพื่อการตัดสินใจ

Key Distinction

EDM A/B INOP ไม่เท่ากับ EDM data หายทั้งหมด; ให้ดู red X และ missing pointer เป็นตัวบอกว่าค่าใด invalid

Key Distinction

EDM FAIL หมายถึงข้อมูล EDM unavailable ทั้งหมด แม้บางข้อความอาจยังแสดงอยู่บน EICAS

5. Note / Caution / Warning

รวบรวมข้อความ safety emphasis ที่เกี่ยวข้องกับ EDM โดยรักษาความหมายตามคู่มือ ไม่ใช้แทน checklist จริง

WARNING — EDM FAIL

เมื่อขึ้น EDM FAIL ให้ถือว่า engine data manager failed และข้อมูล EDM ไม่พร้อมใช้งานทั้งหมด ผู้เรียนต้องจำว่า failure นี้ทำให้สูญเสียการ monitor โดยตรงของ engine, fuel, electrical, hydraulic และ cockpit pressurization

NOTE — EDM A INOP / EDM B INOP

เมื่อมี EDM A INOP หรือ EDM B INOP advisory แสดงอยู่ ให้ถือว่า invalid เฉพาะ parameter ที่ถูกแทนด้วย red X และ/หรือ missing pointer เท่านั้น ไม่ใช่ทุกค่าบน EICAS

NOTE — EDM FAIL Prior to Start

ถ้า EDM FAIL เกิดก่อน start และมี red X ใน IOAT และ ITT counters คู่มือให้ไปอ้างอิงหัวข้อ High IOAT at Start > 80 °C

CAUTION — Training Emphasis

ถ้า EDM FAIL ยังคงแสดงอยู่ อย่าใช้การมีอยู่ของ messages เช่น CKPT PX, CKPT ALT, HYD FL LO, FUEL BAL, L FUEL LO หรือ R FUEL LO เป็นหลักฐานว่าระบบเหล่านั้นยังถูก monitor อย่างน่าเชื่อถือ เพราะคู่มือระบุว่าข้อความเหล่านี้อาจยังแสดง แต่ไม่ monitor ระบบของตนแล้ว

NOTE — PMU Relationship

EDM FAIL ไม่ได้แปลว่า PMU offline โดยอัตโนมัติ; คู่มือระบุว่า PMU should remain online ดังนั้นให้แยก “engine control” ออกจาก “engine data monitoring”

Advisory — Maintenance Before Flight

ถ้า EDM A INOP หรือ EDM B INOP advisory ยังคงอยู่ก่อนบิน คู่มือระบุว่าต้องมี maintenance action ก่อน flight

6. Pilot Mental Model

ใช้ model นี้เพื่อจำบทบาทของ EDM ใน cockpit

PMU

คิดเรื่องการควบคุมเครื่องยนต์/propulsion

EDM

คิดเรื่องการรับข้อมูล ประมวลผล และส่งให้ EICAS

EICAS

คิดเรื่องการแสดงผลให้ pilot เห็นและแจ้งเตือน

7. Quick Check

กดเปิดคำตอบเพื่อทบทวนความเข้าใจ

1) EDM เป็นตัวควบคุมเครื่องยนต์หลักเหมือน PMU หรือไม่?

ไม่ใช่ EDM เป็นระบบ monitor / data manager ส่วน PMU เกี่ยวข้องกับ engine control มากกว่า

2) ถ้าเห็น EDM A INOP แต่ค่า torque ยังไม่มี red X ควรสรุปอย่างไร?

ไม่ควรสรุปว่าข้อมูล EDM ทั้งหมดเสีย ให้ถือว่า invalid เฉพาะ parameter ที่มี red X หรือ pointer หาย

3) ถ้าเห็น EDM FAIL ควรตีความข้อมูล EDM อย่างไร?

ให้ถือว่า EDM data unavailable ทั้งหมด และสูญเสียการ monitor โดยตรงของหลายระบบสำคัญ

4) EDM ส่งข้อมูลไปแสดงที่ใด?

EDM ส่งข้อมูลไปยัง IAC แล้วแสดงบน MFD/EICAS ใน cockpit

5) เหตุใด red X หรือ missing pointer จึงสำคัญ?

เป็น cue ว่าค่านั้น invalid และไม่ควรใช้เป็นฐานในการตัดสินใจ
