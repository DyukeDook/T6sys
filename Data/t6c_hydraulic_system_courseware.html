2) System Overview

ระบบไฮดรอลิกของ T-6C ใช้แรงดันจาก engine-driven pump (EDP) เพื่อขับอุปกรณ์ที่ต้องใช้แรง เช่น flaps, speed brake, nose wheel steering และระบบ landing gear/doors ในเชิง hydraulic consumer โดยในหน้านี้จะไม่ลงรายละเอียดไฟ indicator ของ landing gear

3) Component Breakdown

Reservoir / Power Package

เก็บ fluid และเป็นจุดที่ตรวจ level ผ่าน hydraulic system service bay

Engine-Driven Pump

ปั๊มหลักจากเครื่องยนต์ pressurize normal system และ charge emergency accumulator

Pressure Relief Valve

อยู่ใน main และ emergency system เพื่อป้องกัน damage จาก high system pressure

Emergency Accumulator

เก็บแรงดันสำรองไว้ใช้ emergency extension ของ landing gear/flaps

Check Valve

กัน emergency system bleed back กลับ main system เมื่อ main system fail

Hydraulic Fuse

จำกัด fluid loss เมื่อ emergency side leak เกิน flow ที่กำหนด

Manual Pressure Release Handle

ใช้ release pressurized fluid จาก emergency accumulator กลับ reservoir สำหรับ servicing/checking

HYD SYS Circuit Breaker

เกี่ยวกับ pressure indication และ HYD FL LO sensor; EHYD PX LO sensor ยัง available แม้ CB open

Hydraulic Service Bay

อยู่ lower aft fuselage หลัง right wing trailing edge มี quantity indicator, release handle และ servicing connection

5) Emergency Hydraulic System

หลักการ

Emergency system ไม่ใช่ปั๊มสำรอง แต่เป็น accumulator ที่ถูก charge ไว้ล่วงหน้า โดย normal hydraulic system เมื่อใช้งานฉุกเฉิน emergency accumulator จะปล่อย pressure ผ่าน emergency lines แยกต่างหากเพื่อ extend landing gear และ flaps

Protection logic

Check valve กัน pressure ไหลย้อนกลับ main system และ Hydraulic fuse ช่วยป้องกัน emergency leak ไม่ให้ดึง fluid จาก main system จนหมดเร็วเกินไป

Scenarioผลที่ควรเข้าใจ
Accumulator chargedมี emergency pressure พร้อมใช้ หลัง normal system เคย pressurize accumulator แล้ว
Emergency accumulator pressure ต่ำEICAS แสดง EHYD PX LO; เป็น indication หลักของสถานะ accumulator
Emergency leak > .25 GPMhydraulic fuse จำกัด fluid loss ประมาณ 20–30 cubic inches หรือ 0.5 qt
Emergency leak < .25 GPMอาจค่อย ๆ deplete main reservoir จนเกิด HYD FL LO

7) Note / Caution / Warning

NOTE — Emergency accumulator discharged

ถ้า emergency accumulator ถูก discharge แล้ว EHYD PX LO จะยัง illuminate จนกว่าระบบจะ pressurize อีกครั้ง ต้องแน่ใจว่า light ดับก่อน flight

NOTE — ถ้า hydraulic pressure ต่ำ

Low hydraulic pressure ต่ำกว่า 1800 psi จะทำให้ต้องใช้ emergency gear extension procedure สำหรับ landing gear; สำหรับ flap อาจต้องใช้ emergency landing gear/flap extension system หาก normal pressure ต่ำกว่า usable level

NOTE — Emergency extension consequence

เมื่อ extend ด้วย emergency landing gear extension system แล้ว gear และ flap retraction ไม่สามารถทำได้ จนกว่าจะได้รับการ service/maintenance ตามคู่มือ

CAUTION / EICAS CAUTION — HYD FL LO และ EHYD PX LO

ในเนื้อหาระบบนี้คำว่า caution ส่วนใหญ่เป็น EICAS caution message ไม่ใช่กล่อง CAUTION ของ manual: HYD FL LO อาจบอก fluid leak/quantity ต่ำ; EHYD PX LO บอก emergency accumulator pressure ต่ำ และ emergency landing gear/flap extension system ควรถือว่า inoperative เมื่อ EHYD PX LO illuminate

WARNING

ไม่พบ WARNING block เฉพาะใน hydraulic system description ที่นำมาทำหน้านี้ อย่างไรก็ตาม abnormal/emergency ให้ใช้ checklist/manual ทางการเสมอ เพราะ training aid นี้ไม่ใช่ procedure ควบคุมการบิน

8) Abnormal / Malfunction Mental Model

HYD FL LO

คิดถึง reservoir quantity ต่ำ หรือ leak ในระบบ ถ้าร่วมกับ pressure ต่ำกว่า 1800 psi ให้ถือว่า normal operation ของ landing gear, flaps, speed brake และ NWS ไม่พร้อมใช้งานตามปกติ

EHYD PX LO

คิดถึง emergency accumulator pressure ต่ำ และควรถือว่า emergency landing gear/flap extension system ไม่พร้อมใช้งานตามเจตนาของ checklist/section malfunctions

Pressure decreasing toward / reads 0 psi

อาจเป็น loss of hydraulic pressure หรือ engine-driven hydraulic pump failure/partial failure โดยเฉพาะถ้าไม่มี HYD FL LO หรือ EHYD PX LO illuminate

Sensor / CB issue

ถ้า hydraulic pressure transmitter fail อาจอ่าน 0 psi หรือ abnormal indication แต่ hydraulic systems อาจยังทำงานปกติได้; ถ้า HYD SYS CB open จะไม่มี pressure indication และ HYD FL LO sensor แต่ EHYD PX LO sensor ยังมีอยู่

Do not create your own procedure

หน้านี้อธิบาย logic เท่านั้น ไม่ใช่ checklist ปฏิบัติการ เมื่อเกิดจริงให้ใช้ Flight Manual / checklist ทางการ

10) Quick Check Quiz

1. Normal hydraulic pressure ประมาณเท่าไร?

3000 ±120 psi

2. ถ้า hydraulic pressure ต่ำกว่า 1800 psi cockpit indication เปลี่ยนอย่างไร?

Hydraulic pressure display เปลี่ยนจาก white เป็น amber

3. HYD FL LO หมายถึงอะไร?

Hydraulic reservoir fluid level ต่ำกว่าประมาณ 1 quart หรือ 55 cubic inches

4. EHYD PX LO บอกอะไร?

Emergency accumulator pressure at/below ประมาณ 2400 psi

5. Emergency hydraulic system มี pump สำรองไหม?

ไม่ใช่ pump สำรอง แต่ใช้ emergency accumulator ที่ถูก charge ไว้ล่วงหน้า

6. Hydraulic fuse ทำหน้าที่อะไร?

จำกัด fluid loss เมื่อ emergency hydraulic system leak เกินประมาณ .25 GPM
