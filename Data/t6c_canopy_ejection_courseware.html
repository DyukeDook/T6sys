1) Canopy System

หน้าที่หลัก: ปิด/ปกป้อง cockpit, ช่วย defog/ventilation, และเป็นส่วนหนึ่งของ escape path เมื่อใช้ CFS/ejection seat

System Overview

  • Canopy รวม windscreen เป็นชุด lockable canopy แบบเปิดด้านข้าง
  • ไม่ใช่ canopy แบบ jettisonable; เวลา ejection ใช้ CFS fracture transparency
  • Oil-filled spring struts ช่วยเปิด/ปิดโดยไม่ต้องใช้ power assist
  • ระบบ latch ใช้ overcentering hooks 5 ตัว ขับด้วย continuous drive rod
  • Electrical sensing ใช้ microswitches และ canopy sill switch เพื่อตรวจ locked/latched condition

2) Canopy Fracturing System (CFS)

หน้าที่หลัก: สร้างทางออกให้ pilot ระหว่าง ejection หรือ emergency ground egress โดย fracture canopy transparency ด้วย explosive cord

Components

Componentหน้าที่
FLSC - Flexible Linear Shaped Chargeติดตั้งที่ front transparency รอบ periphery และ centerline เพื่อตัดเป็นสอง halves
MDC - Mild Detonating Cordติดตั้งที่ rear transparency แยกเป็น periphery และ centerline diamond pattern
Internal CFS handlesอยู่หลัง PCL ในแต่ละ cockpit ใช้ actuate transparency ของ cockpit นั้น
External CFS handles/lanyardsอยู่หลัง ground egress doors ซ้าย/ขวา ใช้ actuate ทั้ง front และ rear transparencies
Piezo / flash lamp / laser / fiber optic pathส่งพลังงานไป optical detonators เพื่อเริ่ม detonation chain

Mode of Operation

AutomaticSeat motion activates CFS through seat/rail initiator during ejection
→
Timed fractureDetonation timed with seat passage so transparency moves outward
→
Backup effectIf CFS fails, seat-mounted canopy breaker helps shatter transparency
Manual internalTurn/pull internal CFS handle; affects respective transparency only
→
Manual externalGround rescue lanyard; affects both transparencies

Pattern / Effect

Front transparency

FLSC รอบ periphery และ centerline ตัด transparency เป็น 2 halves แล้วแรง detonation พาชิ้นส่วนออกจาก canopy frame

Rear transparency

Rear ejection detonate เฉพาะ centerline diamond pattern; ถ้าใช้ rear internal CFS handle หรือ external CFS handle จะ detonate ทั้ง periphery และ diamond pattern

3) Ejection Seat System

หน้าที่หลัก: พา pilot ออกจาก aircraft อย่างเป็นลำดับอัตโนมัติ หลังดึง ejection control handle

Seat Overview

  • ใช้ Martin-Baker ejection seats
  • แต่ละ cockpit มี seat ของตัวเอง
  • Seat เป็น fully automatic หลัง ejection initiated
  • ให้ safe escape ได้ตั้งแต่ zero altitude / zero speed ถึงประมาณ 35,000 ft และประมาณ 370 KIAS ภายใน envelope
  • มี Manual Override Handle (MOR) สำหรับ restart seat/pilot separation sequence ถ้า automatic system ไม่ทำงาน หรือใช้ตามเงื่อนไขที่ manual ระบุ

4) Interseat Sequencing System (ISS)

หน้าที่หลัก: กำหนดว่า ejection จะเกิด seat เดียวหรือสอง seat และจัดลำดับการออกจาก aircraft เมื่อต้อง eject ทั้งสองที่นั่ง

ISS Modeผลของการดึง handleข้อจำ
SOLONo gas transfer between seats; seat ที่ initiate จะ eject เท่านั้นใช้เมื่อบิน solo
BOTHทั้งสอง seats eject; rear seat ejects first regardless of which seat initiatesมี separation delay ระหว่าง rear และ front
CMD FWDFront cockpit initiates both seats; rear seat first. Rear cockpit initiates rear seat onlyเหมาะกับ crew concept ที่ให้ front cockpit command dual ejection

5) NOTE / CAUTION / WARNING

กล่องด้านล่างเป็นสรุป/แปลไทยจาก manual เพื่อการเรียน ไม่ใช่ full procedure หรือ wording ทางการ ให้ตรวจ manual/checklist ก่อนใช้งานจริง

Canopy Related

WARNING

Ground egress doors ต้อง unlocked ระหว่าง ground และ flight operations เพื่อให้ ground personnel สามารถ operate CFS ได้เมื่อจำเป็น

CAUTION

เมื่อเปิด canopy ให้วางนิ้ว/หัวแม่มือไว้ด้านนอก canopy rail เพื่อหลีกเลี่ยงการบาดเจ็บ เพราะมีช่องว่างน้อยระหว่าง canopy rail กับ canopy breaker tool bar ของ front cockpit ejection seat

CAUTION

การลด canopy ลงโดยที่ external handle ไม่อยู่ full OPEN หรือ internal handle ไม่อยู่ fully aft อาจทำให้ canopy hooks เสียหาย ต้องมั่นใจว่า rail clear และ pin box closed

CAUTION

ห้ามหมุน canopy handle เร็วหรือใช้แรงมากเกินไป เพราะอาจทำให้ canopy locking system เสียหายหรือทำให้ cam mechanism ของ hooks ทำงานผิดพลาด หากรู้สึก resistance มาก ให้เปิด canopy แล้วตรวจ hooks

WARNING

ให้มี occupant เพียงคนเดียว operate canopy handle ระหว่างปิดจากด้านใน เพื่อหลีกเลี่ยงการหนีบนิ้วหรือมือ

NOTE

Canopy ถูก hold open ด้วย canopy lock ต้อง release canopy lock ก่อนปิด; canopy lock release handle อยู่ที่ right side panel ในแต่ละ cockpit

NOTE

สามารถจัดตำแหน่ง canopy hooks ด้วยมือเพื่อให้เข้า hook receivers ได้ แต่ถ้า hook alignment ผิดปกติต้อง annotate ใน aircraft forms และ aircrew ต้องตรวจว่า internal canopy handle lock canopy closed ได้จริง

NOTE

ถ้า canopy ไม่สามารถ lock จากด้านในได้ ห้าม accept aircraft for flight; หากใช้ crew chief ช่วยปิด/lock canopy aircrew ต้อง re-open และ close canopy จากด้านในอีกครั้ง

CFS / Emergency Ground Egress Related

WARNING
  • CFS internal initiators มี safety pin ต้อง remove/stow before flight และ re-install after flight
  • CFS เป็น explosive canopy fracturing system; detonation ทำให้ shards/fragments กระเด็นออกด้านนอก canopy อาจทำให้บาดเจ็บรุนแรง
  • Ground rescue personnel ควร face away จาก canopy ขณะ activate CFS
  • Aircrew ต้อง oxygen mask on และ visor down ก่อน actuate CFS เพื่อป้องกัน shards และ hot fragments
  • หากทราบหรือสงสัยว่า CFS handle ถูก actuated ห้ามปิด canopy จนกว่าจะ inspect CFS donor assemblies/firing plungers
CAUTION

เพื่อหลีกเลี่ยง accidental CFS activation ต้องมั่นใจว่า area รอบ CFS handle clear ก่อนออกจาก cockpit

WARNING

External emergency ground egress lanyard อยู่หลัง access door แต่ละด้านของ aircraft; door มี lock เพื่อป้องกัน unauthorized access และต้อง remain unlocked ระหว่าง flight/ground operations

NOTE

Internal front/rear cockpit CFS handles operate explosives เฉพาะ transparency ของ cockpit นั้น ส่วน external ground egress handle ด้านใดด้านหนึ่งจะ activate explosives ของทั้งสอง transparencies

NOTE

Operation ของ CFS system ไม่ทำให้ ejection seat fire และไม่ป้องกันการ fire ของ ejection seat

Ejection Seat Related

WARNING
  • Emergency escape system มี explosive cartridges และ rocket propellant; inadvertent firing อาจทำให้บุคคลใน/บน/ใกล้ aircraft บาดเจ็บรุนแรงหรือเสียชีวิต
  • Ejection seats ต้อง pinned เมื่อ canopy ไม่ได้ securely down and locked
  • การ operate ejection seat โดย canopy open หรือ unlocked อาจทำให้บาดเจ็บรุนแรงหรือเสียชีวิต
  • Ejection control handle safety pin ใช้ทำให้ seat safe ระหว่าง non-flight operations; pilot(s) รับผิดชอบ remove/install ก่อนและหลัง flight และต้องแน่ใจว่า seat pin fully inserted/seated
WARNING

เมื่อ eject เหนือ mountainous terrain ที่สูงเกิน 8000 ft MSL manual ระบุให้ใช้ Manual Override (MOR) handle เพื่อ manually separate from seat และ deploy parachute

NOTE

Risk chart assumptions: ejection อยู่ภายใน seat performance envelope, pitch/roll/yaw rate ต่ำ, injury probability เพิ่มตาม speed, sled test data ใช้กำหนด risk areas และไม่รวม parachute landing injuries หรือ canopy burns

ISS / Pilot Hookup Related

WARNING

ถ้า ISS mode selector อยู่ BOTH จะมี delay 0.37 s หลัง rear seat eject ก่อน front seat eject; ถ้า SOLO จะ bypass delay นี้ Manual ระบุให้ set SOLO เมื่อบิน solo และเมื่อ SOLO มีเพียง activating seat ที่ eject

WARNING
  • เมื่อ rear cockpit occupied ห้ามบินโดย ISS mode selector อยู่ SOLO เพราะอาจเกิด unsequenced ejection แบบ front seat first และทำให้ rear cockpit occupant บาดเจ็บรุนแรงหรือเสียชีวิต
  • การ fire seat หนึ่งตัวขณะ ISS อยู่ SOLO ไม่ได้ป้องกันอีก seat จากการ fire
  • หาก ISS อยู่ BOTH และมีการดึง unpinned ejection handle ทั้งสอง seats จะ eject แม้อีก seat จะ pinned
  • ใน CMD FWD, front cockpit สามารถ initiate ทั้ง front/rear seats โดย rear seat ejects first แม้ rear seat ejection handle safety pin จะ installed
  • Seat sequencing unpredictable ถ้า ISS mode selector ไม่ locked in a detent
CAUTION

ก่อนปรับ seat height ต้อง clear seat pan และ side consoles จาก belts, cords, connections เพื่อป้องกันความเสียหายต่อ seat หรือ equipment

WARNING
  • หาก ejection handle ไม่อยู่ upright หรือ harness straps กีดขวาง handle จะทำให้ initiate ejection ไม่ได้
  • ระหว่างเข้า cockpit ต้อง monitor head clearance เพื่อหลีกเลี่ยง canopy alignment pins, latch hooks และ external canopy handle
  • Leg restraint garters ต้อง fitted/connected ถูกต้องและ snug รวมถึงเอา slack ออกจาก leg lines มิฉะนั้นอาจ hang-up during egress
  • Lap belt ต้อง route ตาม manual: ความผิดพลาดอาจทำให้ entanglement กับ SSK lines หรือทำให้ anti-G suit สูญเสีย air supply
NOTE

Leg restraint garters ควรอยู่ just above the boot บนแต่ละขาของ pilot

Quick Check Quiz

1) ทำไม T-6C ถึงต้องใช้ CFS แทนการ jettison canopy?

เพราะ canopy เป็นแบบ non-jettisonable; escape path จึงถูกสร้างโดย fracture transparency ด้วย CFS เพื่อให้ seat ผ่านออกไปได้

2) CANOPY warning บอกอะไร?

บอกว่า latch mechanism และ sensing logic ไม่ยืนยันว่า canopy อยู่ในสภาพ closed and locked

3) Internal CFS handle กับ external CFS handle ต่างกันอย่างไร?

Internal handle ในแต่ละ cockpit ใช้กับ transparency ของ cockpit นั้นเท่านั้น ส่วน external ground egress handle ด้านใดด้านหนึ่ง activate ทั้ง front และ rear transparencies

4) ลำดับ ejection seat แบบสั้นที่สุดคืออะไร?

Pull handle → catapult drives seat up rails → CFS fractures canopy → rocket motor fires → drogue stabilizes → personnel parachute deploys → seat/pilot separation

5) เมื่อบินสองคน ทำไมไม่ควรตั้ง ISS เป็น SOLO?

เพราะ SOLO ทำให้ only activating seat ejects และอาจทำให้เกิด unsequenced ejection แบบ front seat first ซึ่ง manual ระบุว่าอาจทำให้ rear cockpit occupant บาดเจ็บรุนแรงหรือเสียชีวิต

6) ประโยคจำง่ายของระบบนี้คืออะไร?

Canopy keeps cockpit closed. CFS clears the path. Ejection seat takes the pilot out.

One-Minute Summary

Canopy

ปิดและ seal cockpit ตรวจ lock/latched ด้วย switches หากไม่ closed/locked จะเห็น red CANOPY warning

CFS

ใช้ FLSC/MDC fracture transparency เพื่อเปิด escape path สำหรับ ejection หรือ ground egress; manual/external/automatic initiation

Ejection

Martin-Baker seat sequence อัตโนมัติ: catapult, canopy fracture, rocket, drogue, parachute, separation โดย ISS จัดลำดับ front/rear seats
