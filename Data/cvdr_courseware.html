ONE-LINE SUMMARY

DTS/DVR คือส่วนที่ช่วยโหลด mission data และบันทึกภาพ/เสียงเพื่อ debrief; IDARS/FDR คือส่วนที่บันทึกข้อมูลเครื่องบินแบบ crash-survivable เพื่อ mishap, maintenance และ tracking.

2) System Overview

ในบทเรียนนี้ “CVDR” หมายถึงแนวคิดรวมของระบบ recorder ที่เกี่ยวกับ cockpit voice/video/data recording ของ T-6C โดยแยกตามชื่อจริงในคู่มือเป็น 2 กลุ่มใหญ่

DTS/DVR — Debrief & Mission Data

ใช้โหลด mission/navigation data ก่อนบิน, ดึง aircraft/mission data หลังบิน, และบันทึกวิดีโอ 4 channels พร้อมเสียง cockpit จาก pilot intercom system.

Mission data Video Audio Debrief

IDARS/FDR — Flight Data & Tracking

เป็น multifunction processing system ที่ทำหน้าที่เป็น crash-survivable FDR และเก็บข้อมูลสำหรับ mishap investigation, ASIP, ENSIP, LESS, maintenance และ aircraft tracking.

FDR ASIP ENSIP LESS

4) Normal Operation Flow

มองเป็น 4 ช่วง: ก่อนบิน → โหลดข้อมูล → ระหว่างบิน → หลังบิน

1

Pre-flight

ใส่ RMMD เข้า DTS/DVR ตามความต้องการ เพื่อเตรียม mission/navigation data

2

Load Data

เข้าหน้า DTS/DVR ผ่าน MFD MENU → LSK L5 แล้วเลือก load options เช่น tactical, user, frequency, configuration, navigation data

3

Record

กด LSK R6 หรือ HOTAS red trigger เพื่อเริ่มบันทึก 4 video channels พร้อม audio channel

4

Post-flight

ดึงข้อมูล aircraft/mission data เพื่อ debrief และ maintenance; ถอด DTS/DVR cartridge ตาม procedure

DTS/DVR Load Options

Optionข้อมูลที่โหลด

USER DATAmission routes, user waypoints, targets
FREQUENCY DATAmission radio/navigation frequencies และ identification features
CONFIGURATION DATAstudent skill level และ A/A, A/G MFD start pages

5) Recording Logic — START / ABORT

เมื่อยังไม่บันทึก

LSK R6 = START

กด LSK R6 หรือ HOTAS red trigger เพื่อเริ่ม recording แล้ว caption จะเปลี่ยนเป็น ABORT

เมื่อกำลังบันทึก

LSK R6 = ABORT

กด LSK R6 ขณะขึ้น ABORT จะหยุด recording แล้ว caption กลับเป็น START

TRAINING NOTE

วิดีโอและเสียงที่ recorded จะถูก time stamped จาก GPS และมี event markers ที่ crew initiate จาก HOTAS. Time counter แสดงที่ด้านล่างของจอในรูปแบบ hh:mm:ssZ สำหรับ Zulu time และแสดง elapsed time เมื่อ recording starts.

7) Abnormal / Failure Behavior

IAC 1 Failure Effect

เมื่อเกิด full IAC 1 failure คู่มือระบุว่า DTS/DVR stops all video and audio recording.

DTS Load/Download Loss

ใน full IAC 1 failure, DTS ไม่มี capability ในการ load/download data ระหว่าง DTS cartridge กับ IAC 1.

Recorder Maintenance Indications

Yellow END, FDR MAINT หรือ FDR FAIL เป็นสิ่งบอก ground crew/maintenance ว่าต้องตรวจ memory/fault/recorder status.

10) Quick Check / Quiz

1. ชื่อระบบที่คู่มือใช้แทน “CVDR” ในบทเรียนนี้คืออะไร?

ตอบ: DTS/DVR สำหรับ data transfer/digital video recording และ IDARS/FDR สำหรับ flight data recording.

2. DVR บันทึก video channels อะไรบ้าง?

ตอบ: 3 front MFDs และ HUD symbols overlayed over HUD camera video รวมเป็น 4 video channels.

3. เริ่ม DVR recording ได้อย่างไร?

ตอบ: กด LSK R6 หรือกด red trigger button บน HOTAS.

4. FDR green MAINT หมายถึงอะไร?

ตอบ: IDARS memory ประมาณ 80% full เพื่อแจ้ง ground crew.

5. FDR amber FAIL หมายถึงอะไร?

ตอบ: recorder failed หรือ requires maintenance.

6. Full IAC 1 failure ส่งผลกับ DTS/DVR อย่างไร?

ตอบ: DTS/DVR หยุด video/audio recording และ DTS ไม่มี capability ในการ load/download data ระหว่าง DTS cartridge กับ IAC 1.
