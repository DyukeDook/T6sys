ภาพรวม HOTAS

HOTAS คือชุดสวิตช์ที่ให้นักบินควบคุมฟังก์ชันสำคัญโดยไม่ต้องปล่อยมือจาก throttle และ stick โดยใน T-6C สวิตช์อยู่ทั้งบน PCL และ control stick grip โดย front/rear cockpit ใช้งานเหมือนกัน เว้นแต่ระบุไว้เป็นพิเศษ

PCL / Throttle Controls

ปุ่มหลักบน PCL ใช้ควบคุม speed brake, communication/intercom, rudder trim และ simulated weapon/HUD functions

ปุ่ม/สวิตช์ชนิด/ตำแหน่งหน้าที่หลักจำง่าย
SPEED BRAKE
Speed Brake Switch		3-position slide: AFT / center off / FWD, spring-loaded กลับกลางเลื่อน AFT เพื่อ extend speed brake และเลื่อน FWD เพื่อ retractหลัง = กาง, หน้า = เก็บ
COM1/COM2
Transmit Switch		3-way momentary toggle, spring-loaded center off, อยู่ inboard face ของ throttle gripโยกขึ้น transmit COM1, โยกลง transmit COM2 ตราบเท่าที่ยังถือสวิตช์ไว้Up = COM1, Down = COM2
ICS / MUTE
Interphone Switch/Mute		Momentary pushbutton บน inside face ของ throttle gripเมื่อ hot mic: mute external incoming audio; เมื่อ VOX ใช้ key interphone และ mute external audio ใน cockpit นั้นคุยในเครื่อง / ปิดเสียงนอก
RUDDER TRIM
Rudder Trim Switch		3-way rocker: left side in / center off / right side in, spring-loaded กลับกลางกดซ้าย trim nose left; กดขวา trim nose right; ปล่อยแล้วค้าง trim position ปัจจุบันกดด้านไหน จมูกไปด้านนั้น
RANGE
Range Select		Momentary pushbutton บน inboard face ของ throttle gripใน A/A master mode + simulated gun selected ใช้ toggle gun-sight range ระหว่าง 700 และ 1400 ftสลับ range A/A gun
CAGE / UNCAGE
Missile Cage/Uncage		Momentary pushbutton บน inside face ของ throttle gripใน A/A master mode + missile selected ใช้ toggle seeker caged symbology ใน HUD ระหว่าง caged/uncaged; basic skill level ไม่มีผลล็อก/ปลด seeker symbol
WEAPON SUB-MODE
Weapon Sub-Mode Select		Momentary pushbutton บน front face ของ throttleเปลี่ยน submode ตาม master mode และ simulated armament: A/A gun สลับ LCOS/CCIL; A/G bomb วน CCIP → CCRP → DTOS → CCIP; A/G rocket/gun ไม่มีผลตามเงื่อนไขที่ระบุปุ่มวน mode อาวุธ
WINGSPAN / MAN DEP
Wingspan / Manual Depression		Potentiometer แบบ up/down, spring-loaded neutral บน front face ของ throttleA/A gun: ปรับ target wingspan 20–200 ft, default 33 ft; A/G MAN: ปรับ manual sight depression -60 ถึง +225 mils, default 160 milsA/A = wingspan, A/G MAN = mil depression
TDC
Target Designator Control		2-axis x/y slew transducer พร้อม pushbutton บน front face ของ throttle gripใช้ designate/segment target symbol และ slew designation point/box ใน NAV หรือ A/G modes ตาม submode เช่น CCIP, MAN, CCRP, DTOSกด = designate/enable, เลื่อน = slew
PCL CUTOFF FINGER LIFTFinger lift ที่ gate ของ PCLยกเพื่อปลด cut-off gate ให้ PCL ขยับไป cutoff; gate มีไว้ป้องกัน engine shutdown โดยไม่ตั้งใจยก gate = เข้า cutoff ได้
PCL FRICTION
Friction Adjust		มีใน forward cockpit เท่านั้นปรับแรงฝืดของ PCL เพื่อให้คันเร่งอยู่ตำแหน่งที่ต้องการปรับความแน่น throttle
SAFETY NOTE

คู่มือระบุว่ามี mishaps จากการยก PCL idle cut-off gate โดยไม่ตั้งใจระหว่างบิน ดังนั้น finger lift ไม่ใช่ปุ่มใช้งานทั่วไป ต้องระวังเป็นพิเศษ

CAUTION-LIKE NOTE

หลัง engine start หาก PCL ถูกเลื่อนไป cutoff โดยไม่ตั้งใจและเครื่องกำลังดับ คู่มือเตือนว่าอย่าพยายาม relight ด้วยการเลื่อน PCL กลับไป idle เพราะอาจเกิดความเสียหายรุนแรงต่อเครื่องยนต์

Flight Stick / Control Stick Grip

ปุ่มบน stick เน้น flight trim, NWS/target designate, master mode และ simulated weapon release/firing

ปุ่ม/สวิตช์ชนิด/ตำแหน่งหน้าที่หลักจำง่าย
TRIM AIL/ELEV
Elevator/Aileron Trim		Momentary 4-way trim, center off บนหัว stickForward = nose down, Aft = nose up, Left = left wing down, Right = right wing down; ปล่อยแล้วค้างตำแหน่ง trim ปัจจุบันดัน trim ไปทิศที่ต้องการให้ aircraft trim
TRIM INTERPT
Trim Interrupt		Pushbutton บน stick gripกดค้างเพื่อ interrupt power ไปยัง trim actuators และ disengage trim actuators; ปล่อยแล้วจ่าย power กลับหยุด trim ทุกแกนชั่วคราว
NWS / TARGET DESIGNATEMomentary pushbutton บน lower forward face ของ stickWeight on wheels: กดเพื่อ engage/disengage NWS และ advisory light; airborne: ใช้ designate current steer-to point และแสดง/ลบ HUD/TSD target designation symbologyบนพื้น = NWS, บนอากาศ = TD
MASTER MODE
Master Mode Selector		Momentary 4-way toggle forward/aft/down/center, spring-loaded center raised positionForward เลือก A/A; Aft เลือก A/G; Down เลือก NAV และแสดง NAV master mode ชั่วคราวบน UFCP; ใช้ cycle weapon/submode ตามเงื่อนไขหน้า A/A, หลัง A/G, ลง NAV
TRIGGER
First Detent DVR / Second Detent Weapon Rls		2-detent trigger switchDetent 1 เริ่ม DVR recording; Detent 2 ใน advanced mode สั่ง simulated gun pod firing เมื่อเงื่อนไขถูกต้อง และแสดง weapon release event marker “T” บน HUD ขณะกดครึ่งแรกอัด, สุดทางยิง gun sim
BOMB / ROCKET RELEASEPushbutton บนด้านบน/ด้านหน้าของ stick gripสั่ง simulated A/G bomb หรือ rocket release ตาม SMS programming, A/G master mode, weight-off-wheels, gear up และ MAS SIM; แสดง event marker “P” บน HUD ขณะกดปุ่มปล่อย bomb/rocket sim
CONTROL PRIORITY

Rear cockpit trim input override front cockpit trim input ทั้ง rudder trim และ aileron/elevator trim ตามที่คู่มือระบุ

PCL จำเป็นกลุ่ม

บิน/สื่อสาร: Speed Brake, COM1/COM2, ICS/Mute, Rudder Trim

Mission/HUD: Cage/Uncage, Weapon Sub-mode, Wingspan/Man Dep, Range, TDC

Stick จำเป็นกลุ่ม

Flight control: AIL/ELEV Trim, Trim Interrupt, NWS/TD

Mission: Master Mode, Trigger, Bomb/Rocket Release

Memory Hook

PCL = “Power + comm + trim yaw + mission cursor”

Stick = “fly trim + select mode + release/fire”

Quick Check

1) บนพื้น ปุ่ม NWS/TARGET DESIGNATE ทำอะไร?

กดเพื่อ engage/disengage nose wheel steering และควบคุม NWS advisory light

2) บนอากาศ ปุ่มเดียวกันทำอะไร?

ใช้ designate current steer-to point และ command HUD/TSD target designation symbology; กดซ้ำเพื่อลบ current target designated condition

3) Trigger detent 1 กับ detent 2 ต่างกันอย่างไร?

Detent 1 เริ่ม DVR recording; Detent 2 สั่ง simulated gun firing เมื่อเงื่อนไขถูกต้อง

4) PCL Range Select ใช้สลับอะไร?

ใน A/A master mode พร้อม simulated gun selected ใช้ toggle gun-sight range ระหว่าง 700 และ 1400 ft

5) Master Mode forward/aft/down คืออะไร?

Forward = A/A, Aft = A/G, Down = NAV
