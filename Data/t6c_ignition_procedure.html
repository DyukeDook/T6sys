Ignition System Procedure

สรุปแบบ training aid: ยุบข้อมูลซ้ำให้เหลือ common rules, แยกสถานการณ์การใช้งาน และเน้น WARNING / CAUTION / NOTE ให้มองเห็นทันที

Source basis: T-6C Flight Manual - Start and Ignition System, Auto Start Operations, Motoring Run Procedure, and Airstart sections. ใช้เพื่อการเรียนระบบเท่านั้น ไม่ใช้แทน checklist/flight manual ฉบับทางการ

1. Mental Model 2. Decision Matrix 3. Procedure Flow 4. W/C/N 5. Memory Hooks

1. Mental Model: เข้าใจ switch ก่อนจำ procedure

IGNITION

NORM = ให้ PMU คุม igniters

ใช้ใน normal ground auto start และ normal operation. PMU จะ energize/de-energize igniters เมื่อ logic ต้องการ

ON = นักบินสั่ง igniters เอง

ใช้เด่นใน airstart และ PMU OFF case เพื่อให้ ignition พร้อมเมื่อ fuel ถูก introduce

STARTER

AUTO/RESET = momentary

โยกแล้ว spring กลับ NORM. ใช้ initiate auto start sequence หรือ reset/abort บางกรณีตามเงื่อนไข

MANUAL = ค้างจนกว่าจะสั่งกลับ

ต้องยกข้าม detent. ใช้ motoring และ PMU OFF airstart. ไม่ spring-loaded กลับเอง

PMU

PMU = ตัวจัด sequence

ใน normal start, PMU คุม starter, boost pump, igniters และ fuel scheduling พร้อม monitor N1, ITT, fuel flow

เมื่อ PMU OFF, นักบินต้องคุม fuel/ignition/starter และ monitor limit เองมากขึ้น

2. Decision Matrix: ตัดข้อมูลซ้ำให้อยู่ในตารางเดียว

Scenario IGNITION STARTER ใครคุม sequence? Key monitor
Ground Auto Start NORM AUTO/RESET PMU คุม starter, boost pump, igniters, fuel ST READY, N1, ITT, fuel flow, oil pressure
Motoring Run NORM MANUAL ไม่เกิน 20 sec นักบินสั่ง starter เพื่อ clear residual fuel / lower ITT Starter duty cycle, ITT trend, prop area clear
PMU NORM Airstart ON AUTO/RESET PMU ช่วย sequencing แต่ pilot ต้อง monitor เอง N1 rise, ITT rise, fuel flow, oil pressure, altitude
PMU OFF Airstart ON MANUAL นักบินคุมเองมากที่สุด โดยเฉพาะ PCL/fuel Fuel flow 75-80 pph initial, ITT peak, N1 acceleration
Immediate Airstart ON after success check/confirm AUTO/RESET ใช้เมื่อเวลา/altitude จำกัด และ PMU NORM N1 rise within 5 sec, ITT within 10 sec after fuel flow

3. Procedure Flow: เขียนเป็น logic ไม่ใช่ checklist เต็ม

Ground Auto Start Logic

PCL to start positionหา ST READY advisory
Confirm ST READY stabilityควรเห็น ST READY ค้างมากกว่า 3 sec ก่อน start
STARTER AUTO/RESETPMU สั่ง starter + boost pump + igniters + fuel
At ~50% N1Starter/igniters de-energize; engine accelerate to idle ~60-61% N1

Motoring Run Logic

PCL OFFหยุด fuel ก่อน
IGNITION NORMไม่ต้องการจุดระเบิดเพิ่ม
Prop area clearรักษาความปลอดภัยภายนอก
STARTER MANUALmotor engine สูงสุด 20 sec แล้วกลับ NORM

Airstart Decision Logic

PMU FAIL ไม่ขึ้น

ใช้ PMU NORM Airstart: IGNITION ON, STARTER AUTO/RESET, PCL IDLE หลัง N1 มากกว่า minimum ที่ procedure กำหนด

→

PMU FAIL ขึ้น

ใช้ PMU OFF Airstart: PMU OFF, IGNITION ON, STARTER MANUAL, ค่อย ๆ advance PCL พร้อม monitor fuel flow/ITT/N1

4. WARNING / CAUTION / NOTE ที่ควรทำให้เด่น

ใช้ปุ่มด้านบนเพื่อกรองเฉพาะระดับที่ต้องการ

W

Abort engine start ถ้ามีอาการ abnormal

ถ้าเกิดอาการผิดปกติระหว่าง start เช่น ITT พุ่ง, ไม่มี light-off, hung/no start, หรือ indication ไม่ปกติ ต้อง abort ตาม procedure และทำ motoring ถ้ามี fuel introduced

W

PCL above IDLE ก่อน N1 stable อาจทำให้ overtemperature รุนแรง

ในการ airstart ห้ามเร่ง PCL เกิน IDLE ก่อน N1 stabilize ประมาณ idle RPM เพราะ fuel flow จะเพิ่มและอาจทำให้ ITT overtemperature รุนแรงมาก

W

Low altitude airstart: อย่าหน่วงการตัดสินใจ eject

Airstart ไม่แนะนำต่ำกว่า 2,000 ft AGL. ถ้าอยู่ต่ำและสถานการณ์ไม่เอื้อ ต้องให้ priority กับ ejection หรือ aircraft recovery ตาม emergency procedure

C

ST READY ต้องค้างก่อน AUTO/RESET

ก่อนเริ่ม ground auto start ควรแน่ใจว่า ST READY advisory ยัง illuminated มากกว่า 3 sec. ถ้า start โดยไม่มี auto start protection อาจสูญเสีย automatic shutdown feature

C

ถ้า ST READY extinguishes ระหว่าง start: PMU auto protection อาจหาย

เมื่อ ST READY ดับระหว่าง sequence, PMU อาจไม่ cut off fuel ให้เพื่อ terminate start. นักบินต้อง abort manually ด้วย PCL OFF หรือ AUTO/RESET ตามตำแหน่ง PCL/IDLE gate

C

PMU STATUS caution: อย่าดำเนิน Engine Start (AUTO) ต่อ

PMU auto abort function อาจ unavailable. ต้องแก้สถานะ PMU ก่อน ไม่ควร continue auto start เมื่อ PMU STATUS ยังอยู่

C

BOOST PUMP ON ต่อเนื่องทำให้ low pressure fuel pump เสียหายได้

หลัง airstart สำเร็จ/landing ต้องจัดการตาม procedure และแจ้ง maintenance ถึงระยะเวลาที่ BOOST PUMP อยู่ ON หาก manual กำหนด

C

PCL ต้อง OFF ก่อน airstart

PCL OFF ช่วย feather propeller, หยุด fuel flow และป้องกัน fuel เข้า combustion chamber ก่อน ignition พร้อม ซึ่งอาจนำไปสู่ hot start

N

IGN SEL = ignition selected/on

จะขึ้นเมื่อ IGNITION switch อยู่ ON หรือ igniters ถูก activate ระหว่าง auto start. ถ้าค้างต่อเนื่องผิดปกติหลังควรดับ ให้บันทึก/แจ้ง maintenance

N

STARTER MANUAL ไม่ spring-loaded กลับ NORM

หลังใช้ MANUAL ต้องสั่งกลับ NORM เอง. หลัง airstart สำเร็จ ถ้า generator reset ไม่ได้ ให้ verify STARTER อยู่ NORM

N

Motoring Run ใช้ clear residual fuel / lower ITT

ทำหลัง aborted start ที่มี fuel introduced และต้องสังเกต starter duty-cycle cool-down period ก่อน attempt ถัดไป

N

PMU auto start monitor

ในการ ground auto start, PMU monitor N1, ITT และ fuel flow. ใน flight mode, sequencing ยังช่วยได้ แต่ automatic shutdown feature ไม่เหมือน ground start จึงต้อง monitor เองอย่างใกล้ชิด

5. Memory Hooks

จำ switch

  • IGNITION NORM = PMU คุมไฟ
  • IGNITION ON = pilot สั่งไฟ
  • STARTER AUTO/RESET = momentary / auto sequence
  • STARTER MANUAL = ค้างจนกว่าจะกลับ NORM

จำ scenario

  • Ground Auto Start: NORM + AUTO/RESET
  • Motoring: NORM + MANUAL
  • PMU NORM Airstart: ON + AUTO/RESET
  • PMU OFF Airstart: ON + MANUAL
