# T6Sys Study Library

หน้านี้ใช้รวมไฟล์บทเรียน `.html` ที่อยู่ในโฟลเดอร์ `Data` ให้เป็นสารบัญสำหรับเปิดอ่านทีละไฟล์

## วิธีเปิดอ่าน

ดับเบิลคลิก `Open-Reader.cmd`

คำสั่งนี้จะอัปเดตรายการไฟล์จากโฟลเดอร์ `Data` แล้วเปิด `index.html` ให้ทันที

## วิธีใช้งาน

เลือกหัวข้อจากเมนูด้านบนแล้วกด `เปิดหัวข้อ` หรือคลิกหัวข้อจากรายการด้านล่าง หน้าเว็บจะเปิดไฟล์ HTML ในโฟลเดอร์ `Data` โดยตรง

## วิธีเพิ่มไฟล์ใหม่

1. วางไฟล์ `.html` ใหม่ลงในโฟลเดอร์ `Data`
2. ดับเบิลคลิก `Open-Reader.cmd` อีกครั้ง
3. ไฟล์ใหม่จะเข้ามาในเมนูและรายการอัตโนมัติ

ถ้าต้องการอัปเดตรายการอย่างเดียว ให้รัน:

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File .\Update-Reader.ps1
```
